# Manga Translator Studio

本專案是一款「本機漫畫翻譯嵌字工具」。

---

## 功能

- 漫畫文字區偵測
- OCR 文字辨識
- 使用 OpenAI / Gemini API 翻譯（使用者自備 API key）
- 自動嵌字
- 手動調整文字
- 匯出 PNG

---

## 特點

- 完全本機運行
- 不需登入
- 不使用雲端服務
- 支援自訂術語規則（未來）

---

## 開發環境

- Python 3.10+
- Windows（優先支援）

---

## 安裝

```bash
pip install -r requirements.txt
